Required Columns
  • username - Unique username for login
  • email - Email address
  • display_name - Full name of the user
  • role - One of: Student, Faculty, Dean, Coordinator, Admin
Password Information
  • password (Optional) - If column exists and has a value, that password will be used
  • If empty or column missing: Default password "Edulytics@cca" will be used
  • For updates: Use "***EXISTING***" to keep the current password unchanged
Optional Columns (Role-specific)
  • For Students: student_number, course, section
  • For Faculty/Dean/Coordinator: institute
